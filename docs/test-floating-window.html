🪟 Test de Ventana Flotante para Timers

📋 Instrucciones de Prueba:

  • 📌 Botón Flotante: Haz clic en el icono 📌 para abrir la ventana flotante interna
  • 🎮 Controles: Usa los botones de la ventana flotante para controlar el timer
  • 🔄 Sincronización: Los cambios se sincronizan entre ambas ventanas en tiempo real
  • 🪟 Cambio de icono: El botón cambia a 🪟 cuando está activo
  • 🖱️ Arrastrar: Arrastra la ventana por el header para moverla
  • 📏 Redimensionar: Usa el handle en la esquina inferior derecha - el timer se adapta automáticamente
  • ➖ Minimizar: Usa el botón − o doble-clic en el header
  • ⌨️ Atajos: Espacio (play/pause), Ctrl+R (reset), Escape (cerrar)
  • ❌ Cerrar: Usa la X en la ventana flotante o vuelve a hacer clic en 📌

🔧 Características Implementadas:

  • True Picture-in-Picture: Se mantiene encima de TODAS las aplicaciones
  • Document PiP API: API nativa de Chrome para ventanas flotantes
  • Video PiP Fallback: Canvas-based para navegadores más antiguos
  • Sincronización en Tiempo Real: 100ms de actualización para fluidez
  • Control Bidireccional: Controla desde cualquier ventana
  • Detección de macOS: Advertencias específicas para limitaciones
  • Atajos de Teclado: Navegación rápida con teclado
  • Gestión de Memoria: Limpieza automática al cerrar
  • Estado Visual: Indicadores de estado y animaciones
  • Múltiples Fallbacks: Garantiza compatibilidad en todos los navegadores
  • Redimensionamiento Adaptativo: El timer se ajusta automáticamente al tamaño de la ventana

⚠️ Limitaciones en macOS:

  • Aplicaciones en Pantalla Completa: Las ventanas flotantes pueden no aparecer encima de aplicaciones en fullscreen debido a los "Spaces" de macOS
  • Soluciones:
    • Sal de pantalla completa con ⌘+F o Esc
    • Usa aplicaciones en modo ventana normal
    • La ventana aparecerá cuando cambies entre aplicaciones
    • En Mission Control, arrastra la ventana del timer a "Todos los escritorios"
  • Funcionamiento Normal: La ventana flotante sí aparece encima de aplicaciones en modo ventana
